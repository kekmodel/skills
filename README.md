# Claude Code Skills

Claude Code 커스텀 스킬 저장소입니다. 플러그인 마켓플레이스로 등록하여 어떤 컴퓨터에서든 명령어 한 줄로 설치할 수 있습니다.

## 설치

Claude Code에서:

```
/plugin marketplace add kekmodel/skills
```

그 다음 원하는 스킬 묶음을 설치:

```
/plugin install python-dev@kekmodel-skills
```

## 스킬 목록

### python-dev

| 스킬 | 설명 |
|------|------|
| `uv-env-setup` | Python 가상환경 자동 설정 (uv 기반) |
